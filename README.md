# 🛠️ QuizForge: AI Adaptive MCQ Engine

**QuizForge** is a private, high-performance study tool designed to turn any topic, exam syllabus, or reasoning concept into a personalized learning journey.

## ✨ Key Features
- **Adaptive Mastery:** Difficulty scales dynamically (Levels 1–7) based on your performance.
- **Smart Memory:** Uses composite fact-mapping to ensure you never see the same fact twice once mastered.
- **Dynamic Context:** Analyzes your input to generate specific content for Competitive Exams (UPSC, JEE, etc.), Reasoning Scenarios, or English Comprehension.
- **PWA Ready:** Install it on your mobile device for an app-like experience.
- **Privacy First:** API keys and progress data are stored exclusively in your browser's local storage.

## 🔑 Setup
To use this engine, you will need an API key from one of the supported providers:
- [Google Gemini](https://aistudio.google.com/app/apikey) (Free Tier)
- [Groq](https://console.groq.com/keys) (Lightning Fast)
- [OpenAI](https://platform.openai.com/api-keys)

## 📱 Mobile Installation
1. Open the Live Demo link in your mobile browser.
2. **iOS:** Tap `Share` > `Add to Home Screen`.
3. **Android:** Tap `⋮` > `Install App`.
